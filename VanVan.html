The number \(f(c)\) is a

  • local maximum value of \(f\) if \(f(c)\geq f(x)\) when \(x\) is near \(c\).
  • local minimum value of \(f\) if \(f(c)\leq f(x)\) when \(x\) is near \(c\).
In Definition 2 (and elsewhere), if we say that something is true near \(c\), we mean that it is true on some open interval containing \(c\). (p.276, def.2)

This absolute maximum is not a local maximum because it occurs at an endpoint. (p.277, line -7)

A critical number of a function \(f\) is a number \(c\) in the domain of \(f\) such that either \(f'(c)=0\) or \(f'(c)\) does not exist. (p.280, def.6)
